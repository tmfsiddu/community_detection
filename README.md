## 1. NetworkX Basics

Notebook:

```text
01_networkx_basics.ipynb
```

Topics covered:

* Graph creation
* Adding nodes and edges
* Directed and weighted graphs
* Graph visualization using matplotlib
* Graph layouts
* Adjacency matrix
* Degree matrix
* Laplacian matrix
* Shortest paths
* Centrality measures
* Bridges and connected components
* Cliques and maximal subgraphs
* Spanning trees
* Eulerian graphs
